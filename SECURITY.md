# Security Policy

GCRM (Global Conflict Risk Monitor) is proprietary software of RAiTHE INDUSTRIES
INCORPORATED. This repository is source-available for reference; all rights are
reserved.

## Reporting a Vulnerability

Please report suspected vulnerabilities **privately** — do not open a public
issue for a security-sensitive report.

- Preferred: this repository's **Security → Report a vulnerability** (GitHub
  private vulnerability reporting is enabled).
- Alternatively: email **robert@raithe.ca**.

You can expect an initial acknowledgement within a few business days. If a report
is accepted, the fix is deployed to the live service and the reporter is credited
unless anonymity is requested; if declined, you will receive the reasoning.

## Supported Versions

GCRM is delivered as a single continuously-deployed service. Only the current
`main` branch — the live deployment — is supported; there are no back-supported
release lines.
